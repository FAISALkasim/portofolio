# Personal Portfolio Website

This is my **first portfolio project** built using **HTML and CSS**. The website includes a homepage and a contact page to showcase basic web development skills and simple website structure.

## Project Overview

The goal of this project is to practice the fundamentals of front-end web development including:

* Structuring webpages using HTML
* Styling webpages with CSS
* Creating multiple pages and linking them together
* Building a simple responsive layout

## Pages Included

### 1. Home Page (portfolio.html)

The homepage introduces the portfolio and may include:

* Introduction/About section
* Basic information about me
* Links to other pages

### 2. Contact Page (contact.html)

The contact page allows visitors to get in touch and includes:

* Contact information
* A simple contact form
* Navigation back to the homepage

## Technologies Used

* HTML5
* CSS3

Some improvements planned for the future:
